

    Большое количество шоколада, содержащегося в конфетной полости, можно превратить в бутылочки с горячим шоколадом, которые при употреблении могут предотвратить худшие последствия бессонницы, такие как появление фантомов.








    Мороженое, находящееся в конфетной полости, бывает трёх видов: ванильное, шоколадное и ягодное. Шариками из этих блоков можно швыряться, как снежками. Сделанные из молока, при попадании в цель они будут снимать эффекты зелий.
    Все сладкие продукты, находящиеся в конфетной полости, имеют очень малую вероятность вызвать эффект сахарной лихорадки при употре- блении. Это значительно сокращает время для потребителей этих продуктов и позволяет им атаковать быстрее на очень короткое время. Вероятнее всего получить сахарную лихорадку, съев пломбир.









    Леденцовые трости — это обычный предмет из в конфетной полости, используемый во множестве рецептов. Их можно сочетать с ледомятой, которая обладает высокой реакционной способностью, для создания ледомятных копий.





    Эти копья заморажива- ют цель при попадании, подобно рыхлому снегу. Кроме того, при ударе они наносят урон по определённой области.
    Ледомята взрывоопасно реагирует с фиолетовой газировкой, и если падающий блок ледомяты коснётся жидкости, он взорвётся. Эта реакция может быть использована для создания ракет из фиолетовой газировки, которыми можно заменить фейерверки для полёта на элитрах.
