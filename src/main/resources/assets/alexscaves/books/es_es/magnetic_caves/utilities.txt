

    A partir de lingotes de neodimio se pueden fabricar imanes azures y escarlatas.







    Estos imanes son muy poderosos, ya que los de color azul pueden alejar los bloques magnéticos de ellos, y los de color escarlata pueden atraerlos hacia ellos.
    Un imán puede mover hasta 27 bloques adyacentes a la vez, siempre que tengan suficiente espacio libre y sean todos metálicos o estén conectados mediante un bloque de slime.
    Estos imanes tienen un alcance limitado, generalmente de solo unos pocos bloques. Este alcance se puede aumentar utilizando el lingote de neodimio respectivo en el bloque, y reducir utilizando el lingote opuesto.

    El telenúcleo, obtenido de un teletor, se puede utilizar para crear nuevas bombillas de Tesla combinándolo con galena cargada.








    También se puede usar para crear una poderosa herramienta conocida como §lguante §lde §lgalena§r.






    El guante de galena se puede usar en la mano secundaria mientras se sostiene una herramienta magnética, como un hacha de hierro, en la mano principal. Al usarse, lanzará la herramienta hacia adelante, hasta unos 20 bloques. La herramienta dañará a cualquier criatura contra la que se golpee, o comenzará a minar cualquier bloque si es del tipo adecuado. El guante se gastará lentamente durante el uso, pero se puede reparar con un telenúcleo.
    Además de usarse para hacer el guante de galena, las bolas de ferroslime se pueden usar para hacer virutas de metal, que luego se pueden fundir para obtener chatarra. También se pueden usar para preparar una poción de magnetización, que otorga la capacidad de reaccionar a fuerzas magnéticas sin usar armadura de metal.











    Los notificadores se pueden combinar con un telenúcleo y un lingote de neodimio azur para crear un §lholocodificador§r. Este objeto se puede vincular a cualquier criatura al usarse sobre ella. Cuando se coloca en un §lproyector §lholográfico§r, se mostrará un holograma de la criatura vinculada. Si no está vinculado, se mostrará en su lugar el usuario del holocodificador.











    Los notificadores también se pueden combinar con un lingote de hierro para crear cuatro luces magnéticas.
    El peso pesado, obtenido de los dividroides, se puede usar para crear un §ldemoledor §lde §lcantera §lmagnética§r.







    Este dispositivo solo es útil cuando se combina con cuatro luces magnéticas y la propia §lcantera §lmagnética§r.






    Cuando se coloca una cantera magnética, los límites del área que se va a explotar automáticamente se pueden marcar creando un rectángulo con cuatro luces magnéticas detrás del bloque de cantera magnética. Una vez que se vea un rayo entre las cuatro luces, se puede colocar el demoledor dentro de estos límites para activar la cantera.
    La cantera seguirá extrayendo todos los bloques rompibles que se encuentren debajo de su área designada y, una vez destrozados, los bloques saldrán volando por la parte superior del bloque principal. Una vez que no haya más bloques para explotar, la cantera quedará inactiva.



    El corazón de hierro del magnetrón también tiene un uso más: se puede transformar en un §lescudo §lresistor§r.






    El escudo resistor se puede utilizar como un escudo normal, excepto que crea un pulso magnético mientras se utiliza que daña a los enemigos cercanos. La polaridad de este pulso se puede cambiar al agacharse mientras se usa el escudo. Si se utiliza el lado azur, el pulso repelerá a las entidades afectadas, y si se utiliza el lado escarlata, las atraerá.
